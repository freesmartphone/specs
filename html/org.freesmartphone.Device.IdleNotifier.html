freesmartphone.org Framework DBus Interface Specification

org.freesmartphone.Device.IdleNotifier

Description

The Idle Notification Interface provides information about the device's idle state.

Namespace

org.freesmartphone.Device.IdleNotifier

Methods

  • GetState
  • SetState

Signals

  • State

Errors

None

Methods

GetState ( s )

Description: Get the current idle status.

Parameters

s: status

The idle status. Expected values are:

  • "BUSY"
  • "IDLE"
  • "IDLE_DIM"
  • "PRELOCK"
  • "LOCK"
  • "SUSPEND"
  • "AWAKE"

SetState ( ) → s

Description: Force an idle status.

Returns

s: status

The idle status. See 'GetState' for valid values.

Signals

State ( s )

Description: Sent, when the idle status has changed.

Parameters

s: status

The idle status. See 'GetState' for expected values.
