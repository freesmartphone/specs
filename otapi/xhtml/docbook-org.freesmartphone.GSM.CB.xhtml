GetCellBroadcastSubscriptions ()
GetCellBroadcastSubscriptions (out 's' channels)
Get current cell broadcast subscription channels.
channels:
The channels you are subscribed to. Simple strings to expect are
          "none" and "all". Also valid are compound descriptions of channel lists like
          "1,2,3-5,7,234,10-5".
        
Implementation notes:
This maps to the GSM 07.07 command +CSCB=?, see 3GPP TS 07.05 Chapter 3.3.4.
SetCellBroadcastSubscriptions ()
SetCellBroadcastSubscriptions (in  's' channels)
Set cell broadcast subscription channels.
channels:
The channels you want to subscribe. Valid simple strings are
          "none" and "all". Also valid are compound descriptions of channels lists like
          "1,2,3-5,7,234,10-5".
        
Implementation notes:
This maps to the GSM 07.07 command +CSCB=(...), see 3GPP TS 07.05 Chapter 3.3.4.
