GetAuthStatus ()
GetAuthStatus (out 's' status)
Retrieve the authentication status for the SIM card.
status:

          The authentication status for the SIM card. Values to expect:
          "READY" = ME is not pending for any password,
          "SIM PIN" = ME is waiting for SIM PIN to be given,
          "SIM PUK" = ME is waiting for SIM PUK to be given,
          "SIM PIN2" = ME is waiting for SIM PIN 2 to be given,
          "SIM PUK2" = ME is waiting for SIM PUK 2 to be given.

GetSimInfo ()
GetSimInfo (out 'a{sv}' info)
Get information about the IMSI, the subscriber numbers,
        and the country code of the SIM card.
info:

          Information about the SIM card. Required tuples are:
          ("imei", string)
          Optional tuples are:
          ("subscriber_numbers", array),
          ("dial_prefix", string),
          ("country", string).
        
Implementation notes:

        This can map to the following GSM 07.07 commands:
          +CIMI (see 3GPP TS 07.07 Chapter 5.6),
          +CNUM (see 3GPP TS 07.07 Chapter 7.1).
          The dial code is computed by cross-referencing ITU E.212 (Land Mobile Numbering Plan).

SendGenericSimCommand (in  's' command,
                       in  's' result)
Send a generic SIM command to the SIM card.
command:
The command to send, encoded as described in GSM 11.11.

The result of the command, encoded as described in GSM 11.11.
Implementation notes:

SendRestrictedSimCommand ()
SendRestrictedSimCommand (in  'i' command,
                          in  'i' fileid,
                          in  'i' p1,
                          in  'i' p2,
                          in  'i' p3,
                          in  's' data,
                          in  's' result)
Send a restricted SIM command to the SIM card.
command:

          The command to send. Valid values are:
          176 = READ BINARY,
          178 = READ RECORD,
          192 = GET RESPONSE,
          214 = UPDATE BINARY,
          220 = UPDATE RECORD,
          242 = STATUS.
        
fileid:
The identifier of an elementary datafile on SIM. Mandatory for every command except STATUS.
p1:
Parameter 1 passed to the SIM. Mandatory for every command except STATUS and GET RESPONSE.
p2:
Parameter 1 passed to the SIM. Mandatory for every command except STATUS and GET RESPONSE.
p3:
Parameter 1 passed to the SIM. Mandatory for every command except STATUS and GET RESPONSE.
data:
The command data to the SIM, encoded as described in GSM 11.11.
result:
The result of the command, encoded as described in GSM 11.11.
Implementation notes:
This maps to the GSM 07.07 command +CRSM=(command,fileid,p1,p2,p3,data), see 3GPP TS 07.07 Chapter 8.18.
GetHomeZones ()
GetHomeZones (out 'a(siii)' homezones)
Retrieve the homezones coordinates, if stored on SIM.
homezones:

          An array containing up to four homezones in Gauss-Krüger coordinates. The array contains of four-tuples with:
          (name:string), the name of the zone,
          (x:int), the X value of the zone center in Gauss-Krüger format,
          (y:int), the Y value of the zone center in Gauss-Krüger format,
          (r:radious), the R*R value of the zone dimension in meters.
        
Implementation notes:
This maps to the GSM 07.07 command +CRSM=176,28512,0,0,123, see 3GPP TS 07.07 Chapter 8.17.
GetPhonebookInfo ()
GetPhonebookInfo (out 'a{sv}' info)
Request information about the SIM phonebook.
info:

        Information about the SIM phonebook storage. Tuples to expect:
        ("slots", int:value) = maximum of entries stored on the SIM,
        ("used", int:value) = actual number of entries stored on the SIM,
        ("number_length", int:value) = maximum length for the number,
        ("name_length", int:value) = maximum length for the name.
      
Implementation notes:
This maps to the GSM 07.07 command +CPBR=?, see 3GPP TS 07.07 Chapter 8.12.
